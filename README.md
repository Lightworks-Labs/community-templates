# Lightworks Community Templates

A collection of GitHub Actions and templates for use with [Lightworks](https://app.lightworks.md).

## Available Actions

### [search-indexer](./search-indexer)

Build search indexes from markdown files for fast full-text search in Lightworks.

```yaml
- uses: Lightworks-Labs/community-templates/search-indexer@v1
```
